# Replay CLI

Install the Replay CLI to browse and upload your AI coding sessions.

## Install

```sh
curl -sSfL https://install.replay.md | sh
```

### Options

Install a specific version:

```sh
VERSION=0.1.0 curl -sSfL https://install.replay.md | sh
```

Install to a custom directory:

```sh
REPLAY_INSTALL_DIR=/usr/local/bin curl -sSfL https://install.replay.md | sh
```

## Supported platforms

| OS | Architecture |
|----|-------------|
| Linux | x86_64, aarch64 |
| macOS | x86_64 (Intel), aarch64 (Apple Silicon) |
| Windows | x86_64 (download manually from [releases](https://github.com/skillsynchq/replay-releases/releases)) |
